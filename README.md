# BEON百华山通量站数据诊断分析

## 项目简介

本项目用于对北京园林绿化生态系统监测网络（BEON）百华山通量站的连续观测数据进行质量诊断分析。

## 数据说明

- **数据来源**：百华山通量站
- **时间范围**：2022年 - 2025年
- **采样频率**：30分钟
- **指标数量**：50个气象和通量指标

## 文件说明

```
├── README.md              # 项目说明文件
├── run_simple.py          # 数据诊断Python程序
├── 数据诊断报告.txt        # 诊断结果报告
└── data/                  # 原始数据文件
    ├── baihuashan_flux_2022_1.csv
    ├── baihuashan_flux_2023_1.csv
    ├── baihuashan_flux_2024.csv
    └── baihuashan_flux_2025_1.csv
```

## 诊断指标

### 气象数据
- 空气温度、气压、湿度、风速、风向
- 辐射：净辐射、短波辐射、长波辐射、光合有效辐射
- 土壤：土壤温度、土壤含水量

### 通量数据
- CO2净通量（NEE）
- 总初级生产力（GPP）
- 生态系统呼吸（Reco）
- 蒸散发（ET）

## 使用方法

```bash
python run_simple.py
```

诊断结果将输出到 `数据诊断报告.txt`

| 指标 | 最小值 | 最大值 | 单位 |
|------|--------|--------|------|

| 气压 | 50000 | 110000 | Pa |
| 相对湿度 | 0 | 100 | % |
| 风速 | 0 | 35 | m/s |
| 净辐射 | -200 | 1000 | W/m² |
